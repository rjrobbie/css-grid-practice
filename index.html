How to make a directory

  • cd = changes directory
  • mkdir = makes a new directory
  • rm -r = removes a directory

Committing changes

  • Add the changes to git = git add .
  • Add a message about changes = git commit -m "message goes here"
  • Send changes = git commit

    • Merging branch to main

    1. Open GitHub and merge the changes
    2. Then create new pull request - should see a message at the top the main
    3. Click the green button to create new pull request
    4. Click merge pull request
    5. Finally, merge pull request
    6. git merge (useThisBranch) - in the main branch

    Updating a branch

    Update branch with remote (hosted on local server) git checkout b1 git merge origin/master
